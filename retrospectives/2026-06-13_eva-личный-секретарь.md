# Рефлексия: Ева — личный секретарь (email дайджест)

Дата: 2026-06-13

---

## 1. Какая задача была

Расширить агента Еву до «личного секретаря»: добавить чтение почты через IMAP
и отправку дайджеста писем в Telegram. Поддержать два ящика — Gmail и Яндекс.
Параллельно разобраться с текущим состоянием Евы и что нужно чтобы разблокировать
уже существующий Telegram-дайджест.

---

## 2. Как решали

1. Запустили два Explore-агента параллельно — полная картина eva.py, планов, инфраструктуры
2. Спланировали 5 фаз: разблокировка Telethon → IMAP ридер → email дайджест → scheduled loop → UX
3. Уточнили у Бориса: дайджест нужен в Telegram, не на email (убрали SMTP)
4. Реализовали `tools/email_reader.py`, правки `config.py`, `eva.py`, `main.py`
5. Добавили поддержку двух ящиков: `EMAIL_USER` / `EMAIL_USER_2` + `_email_accounts()`
6. Исправили UnicodeEncodeError для Яндекса: `imap._encoding = 'utf-8'`
7. Задеплоили, получили `[Errno 111] Connection refused` с Railway для обоих ящиков
8. Диагностировали: локально оба ящика работают → проблема в Railway (блокировка порта 993)
9. Изучили документацию Railway — порт 993 (IMAP) заблокирован на Hobby/Free плане
10. Обновили план: следующий шаг — переход на Gmail API + Yandex Mail API (HTTPS)

## 4. Что можно было лучше

- **Проверить ограничения Railway по портам до написания IMAP-кода** — сэкономило бы час работы.
  Достаточно было одного поиска: «Railway outbound port 993».
- Первую проверку `railway run python -c "imap..."` нужно было запустить сразу после деплоя,
  до того как тратить время на диагностику UnicodeEncodeError (который оказался вторичной проблемой).
- `railway run` запускает код **локально** с Railway-переменными, а не на сервере —
  это нужно помнить при диагностике сетевых проблем.

---

## 5. Что узнал нового о проекте и пользователе

- Railway блокирует исходящий порт 993 (IMAP SSL) — это фундаментальное ограничение платформы,
  не обходится без смены плана или хостинга.
- Google и Яндекс дополнительно блокируют IMAP с IP облачных провайдеров — двойная блокировка.
- Борис использует два ящика: Gmail и Яндекс.
- Яндекс генерирует пароли приложений в ASCII (16 символов), UnicodeEncodeError был
  из-за старого кириллического пароля который пользователь ввёл изначально.
- `railway run <cmd>` = локальный запуск с env Railway, не запуск на сервере.
