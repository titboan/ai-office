## Что узнал нового
- Recharts axis ticks (`tick={{ fill: 'currentColor' }}`) позволяют унаследовать CSS `color` из body — не нужны отдельные переменные для каждого чарта.
- Telegram `tg.colorScheme` доступен сразу при `tg?.ready()`, достаточно в одном `useEffect`.
- `bg-red-900/20` — Tailwind поддерживает opacity-суффикс через `/`, работает и с `dark:`.
