# Бэклог нереализованных рекомендаций (анализ 2026-07-03/04)

Статус: в работе

## Контекст

По итогам двух раундов анализа проекта (архитектура/security + продуктовая стратегия) часть
находок уже закрыта в предыдущих сессиях: тесты+CI, дизайн-токены дашборда, owner-only доступ,
алерты фоновых синков, атомарные Redis-локи, approval-gate на PR у Кевина, заморозка Кевина.
Этот план фиксирует всё, что осталось нереализованным, чтобы не потерять и разобрать по
приоритету, а не пытаться сделать всё разом.

Ничего в этом плане не начато. Реализовывать по одному пункту за раз, начиная с Фазы 1.
