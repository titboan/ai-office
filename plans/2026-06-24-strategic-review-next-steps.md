### 🟡 3. Замена генератора изображений у Дэна (MEDIUM EFFORT)
Pollinations.ai = 30–120 сек, поэтому Дэн заморожен. Без него выпадает весь визуальный пайплайн (инфографика, лендинги).

**Варианты:**
- **Replicate / Flux** — $0.003/изображение, ~5 сек → самый дешёвый
- **DALL-E 3** — $0.04/изображение, ~8 сек → лучшее качество
- **Stability AI** — $0.002/step, быстро

**Рекомендую:** Replicate (Flux-dev) — дёшево + быстро + API простой.

## Что НЕ стоит делать сейчас

| Идея | Почему нет |
|------|------------|
| Ева (email) | Нужен VPS для IMAP proxy, low ROI для текущего бизнеса |
| Yandex Market | Третий маркетплейс — слишком рано, WB/Ozon ещё не закрыты полностью |
| Webhook mode вместо polling | Polling стабилен, webhooks добавляют сложность без видимой пользы |
| 1C интеграция | Сложная интеграция, требует онпремис |
| LangGraph/CrewAI | Противоречит принципам проекта |
| Refactor длинных функций | Рабочие функции, риск > польза |
