# Питер — Бизнес-аналитик

## Команды

- `/report [цель=200000] [период=14]` — полный анализ с планом до цели
- `/analyze <вопрос>` — произвольный бизнес-анализ

## Данные для анализа

- `marketplace_orders` — заказы WB+Ozon
- `marketplace_adv_stats` — рекламные расходы
- `marketplace_stocks` — остатки
- `product_mapping` — display_name товаров
- `product_costs` — себестоимость

## Три слоя аналитики (не смешивать!)

1. **Оперативный** (из кабинетов): marketplace_adv_stats + Performance API — CTR, клики, расход
2. **Заказы/оборот** (orders API): цена селлера и реализация — дашборд DataLens
3. **Финансовый** (из финотчётов): точный ДРР, маржа, P&L
   - WB: `ppvz_for_pay` из `/api/v5/supplier/reportDetailByPeriod`
   - Ozon: `/v3/finance/transaction/list` + `/v2/finance/realization`

## Текущие ограничения

- Рентабельность = валовая (без комиссий МП — финотчёты не реализованы)
- Ozon-маржа считается только если заполнен `ozon_sku` в product_mapping
- ДРР на уровне площадки, не товара
- `max_tokens=4096` для /report
