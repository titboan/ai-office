# Устанавливать git config в начале каждой сессии

## Правило

В самом начале каждой сессии (до первого коммита) выполнять:

```bash
git config user.email noreply@anthropic.com
git config user.name Claude
```

## Почему

Без этого коммиты создаются с дефолтным email контейнера, GitHub показывает их как "Unverified",
стоп-хук выдаёт предупреждение. Исправить post-factum без force push в main нельзя.

## Когда обнаружено

Сессия 2026-06-22, SEO-пайплайн. Все 5 коммитов прошли с неправильным email.
