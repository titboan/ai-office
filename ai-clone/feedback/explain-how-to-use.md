После завершения любой технической задачи — добавить в ответ блок «что это даёт и как использовать»: примеры SQL-запросов, команды, конкретные изменения для пользователя.

**Why:** Борис спрашивает это после каждой фичи. Лучше давать сразу, не ожидая вопроса.

**How to apply:** Сразу после отчёта «готово» — короткий раздел с практическими примерами. Особенно важно для БД-изменений, новых таблиц, новых эндпоинтов.
