# Правило: план всегда в `plans/` репозитория

Дата: 2026-06-13

## Что произошло

При работе в plan mode агент создал план в локальной папке Claude Code
(`/root/.claude/plans/...`) вместо `plans/` репозитория.
Борис напомнил — пришлось создавать заново.

## Правило

Для любой задачи, затрагивающей 2+ файла или требующей нового агента/таблицы/эндпоинта:

1. Создать файл `plans/YYYY-MM-DD-название-задачи.md`
2. Формат: фазы с чек-боксами `[ ]` / `[x]`, строка `Статус: в работе`
3. Коммитить вместе с первым изменением кода

Локальная папка `/root/.claude/plans/` — только для plan mode UI, не для хранения планов проекта.

## Как проверить

После ExitPlanMode — убедиться что файл существует в `plans/` репозитория.
